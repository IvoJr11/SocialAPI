Este es un proyecto creado con la finalidad de aprender sobre APIs en .NET7
He tratado de hacer a mano una API de una red social como Twitter, Instagram, etc.
Está desarrollada mediante el método "Code First" utilizando LINQ.

Para utilizarlo, solo deben clonar el proyecto, hacer un build y asegurarse que
las dependencias estén en orden. Además, ya sea desde la terminal o desde el mismo Visual Studio, pueden añadir y actualizar
las migraciones a la base de datos y estaría todo listo!

A fecha de hoy, 22/08/23 sigue en proceso, así que pueden llegar a encontrarse
problemas.

Cabe recalcar que es completamente abierta y cualquier sugerencia es bienvenida,
espero que ayude para programadores que, como yo, quieran aprender este lenguaje,
para ello iré dejando comentarios a lo largo del código explicando distintas porciones
para que sea lo más entendible posible! <3
